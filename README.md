# 🏠 House Price Prediction

A Machine Learning project that predicts house prices based on features like area, BHK, and location.

## 🔧 Technologies Used
- Python
- Pandas, scikit-learn
- streamlit (for GUI)

## 📂 Files
- `house_price_model.pkl`: Trained model
- `House_Price_app.py`: GUI code
- `data.csv`: Dataset
